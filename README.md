### Description

Boom2D is a 2D engine that uses DirectX 8.0, features include:

- Sprites

- ANIMATED sprites

- Coloring sprites

- Collision detection

- Tiles

- Alpha blending

- Rotating sprites

- Scrolling

- Scaling sprites

- Fullscreen + Windowed mode

- Drawing text

- Multiple windows

- Colorkeys on sprites

- And more...

Uses Direct3D.

This also includes a sample program of some stuff you can do with the engine.
